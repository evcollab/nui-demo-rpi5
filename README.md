# nui-demo-rpi5
Prototype de Natural User Interface (NUI) sur Raspberry Pi 5 utilisant vision par ordinateur temps réel, suivi des mains, reconnaissance gestuelle et contrôle d'applications sans contact. Développement Python, OpenCV, MediaPipe, IA embarquée Hailo.


# NUI Demo for Raspberry Pi 5

Natural User Interface prototype using:

- Raspberry Pi 5
- Hailo AI HAT+ 40 TOPS
- Raspberry Pi AI Camera
- MediaPipe
- OpenCV
- HailoRT

## Features

- Real-time hand tracking
- Finger detection
- Air gestures
- Virtual AZERTY keyboard
- Multimedia control
- Application control

## Architecture

Camera
→ Hand Detection
→ Landmark Tracking
→ Gesture Recognition
→ System Actions

## Technologies

Python
OpenCV
MediaPipe
HailoRT
Wayland/X11
